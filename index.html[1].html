[Role & Goal]

너는 중학교 수행평가용 '인터랙티브 디지털 수행평가지'를 제작하는 전문 웹 퍼블리셔다. 나의 요구사항을 반영하여 HTML 파일 하나로 완성되는 인터랙티브 수행평가지를 제작해줘.

[Technical Requirements]

1. html2canvas 라이브러리(CDN)를 사용하여 화면을 이미지로 저장하는 기능을 포함할 것. 

2. 결과물 저장 시 파일명은 지구촌 세계시민 뉴스_학번_이름.pdf로 설정할 것. 

3. 모든 스타일은 <style> 태그 안에, 모든 기능은 <script> 태그 안에 작성하여 HTML 단일 파일로 동작하게 할 것. 

4. 모바일·PC 양쪽에서 모두 보기 좋게 반응형으로 제작할 것.

[Layout & UI Requirements]

1. 상단 고정 영역: - 학번·이름 입력 텍스트 박스 - [나의 활동지 이미지로 저장하기] 버튼 (눈에 잘 띄게 배치) 

2. 입력 칸은 학생이 직접 타이핑할 수 있는 textarea 또는 input 형태로 구성할 것. 

3. 활동지의 각 섹션은 시각적으로 명확히 구분되도록 카드 또는 박스 형태로 디자인할 것.

[Design Concept]

디자인 컨셉: 지구촌에서 일어나는 다양한 문제들을 다루는 신문 기사 형태의 수행평가지 (실제 신문과 같은 배경색과 구성, 중요한 내용이 강조되는 레이아웃, 신문 기사 같은 명확한 폰트 사용)

[Content Specification]

아래 내용을 바탕으로 학습지를 구성해줘.

① 성취기준: [9사(지리)02-02] 종교와 관련된 아시아의 다양한 문화경관과 생활양식을 파악하고, 세계시민으로서 문화 다양성에 대한 이해와 수용성을 높인다.

[9사(지리)03-03] 지역 간 역학관계에 따른 유럽의 통합과 분리의 움직임이 유럽연합의 변화와 주민 생활에 미치는 영향을 탐색한다.

[9사(지리)06-02] 태평양 지역이 겪고 있는 환경 문제를 조사하고 그 해결에 참여할 수 있는 방안을 제안한다.

② 학습 활동 유형: 신문 기사 작성(지구촌의 다양한 지리적 이슈 중 한 가지 선정 → 해당 지리적 이슈가 나타나는 위치 세계지도에 표기 → 해당 지리적 이슈의 원인 작성 → 해당 지리적 이슈가 미친 영향 분석 → 해당 지리적 이슈에 대한 세계시민적 해결 방안(실천 방안) 제안 → 실천방안을 담은 슬로건 또는 로고 제작)

③ 학습지에 포함할 세부 활동 내용:

  1. 지구촌의 다양한 지리적 이슈 중 한 가지 지리적 이슈 선정하기 (수업시간에 배운 것 중에서 한 가지 선정)
  2. 해당 지리적 이슈가 나타나는 국가나 지역의 위치를 세계지도 백지도에 표기하기
  3. 해당 지리적 이슈가 나타나는 자연적 원인 또는 인문적 원인 또는 둘 다로 원인 작성하기
  4. 해당 지리적 이슈가 해당 지역또는 국가, 또는 다른 지역에 미친 영향 분석하기
  5. 해당 지리적 이슈에 대한 세계시민적 해결방안 또는 실천방안 제안하기
  6. 해결방안 또는 실천방안을 실제로 촉구하기 위한 슬로건 또는 로고 제작하기

[Output Format]

- 완성된 HTML 코드 전체를 코드 블록으로 출력해줘. - 별도의 CSS 파일이나 JS 파일 없이 HTML 파일 하나만으로 실행 가능해야 해. - 학생이 바로 사용할 수 있도록 샘플 텍스트나 안내 문구를 적절히 넣어줘.
